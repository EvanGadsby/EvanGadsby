As you can see these courses covers a variety of different skills related to comupter information systems, as well as buisiness related topics. MRU's BCIS degree also includes a COOP term to better help students get some real experience in the work environment. So what kind of careers might would be possible for a BCIS student? well lets take a look!

BCIS Careers

BCIS is a very flexable degree that allows for a wide range of possible careers. Some possible careers are:

  • Systems administrator
  • Network administrator
  • Database adinistrator
  • IT support manager
  • Business analyst
  • Web/mobile application developer
  • Application architect
